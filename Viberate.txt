
Introduction
Welcome to Viberate API, your source of daily-refreshed data on millions of music entities.

We collect data from all relevant streaming and social media channels and use it to map, analyze and rank millions of artists, tracks, playlists, venues, labels, festivals, and events. All our databases are carefully curated, with no double entries.

Check our API documentation on how to best implement our data.

Authorization
You will find API token in your API Managment Page. API key in Header is required for each request. curl -H 'Accept: application/json' -H 'Access-Key: {API token}' {URL}

Rate Limiting
Rate limits depend on your API package. You can find detailed package descriptions and prices here.
